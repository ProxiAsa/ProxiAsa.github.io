Infrastructure Complète d'Entreprise

Conception et déploiement d'une infrastructure d'entreprise complète intégrant pare-feu, serveur web, base de données, annuaire Active Directory, stockage NAS et ERP Odoo. Segmentation réseau par VLAN avec politique de sécurité unifiée.

Réalisé
📅 2025
🏫 Projet scolaire BTS SIO
pfSense Active Directory Odoo NAS Synology PostgreSQL Nginx VLAN Linux Debian

Contexte

Projet de fin d'année BTS SIO : simuler le déploiement d'une infrastructure informatique complète pour une PME fictive. L'objectif était de couvrir l'ensemble du périmètre SI : sécurité réseau, services d'annuaire, hébergement web, stockage, base de données et ERP métier, avec une segmentation réseau par VLAN.

Périmètre

Infrastructure complète PME fictive (~50 postes)

Virtualisation

Proxmox VE — toutes les VMs sur un cluster local

Réseau

Segmentation VLAN, pare-feu pfSense, routage inter-VLAN

Composants de l'infrastructure

Pare-feu & Routeur
pfSense
Filtrage du trafic entrant/sortant, routage inter-VLAN, règles de firewall par zone, VPN site-à-site, portail captif Wi-Fi invités.
Annuaire d'entreprise
Active Directory — Windows Server 2022
Gestion des utilisateurs, groupes et ordinateurs. GPO déployées par OU. DNS et DHCP intégrés. Authentification centralisée pour tous les services.
Serveur Web
Nginx — Debian 12
Hébergement du site vitrine et de l'interface web d'Odoo. Reverse proxy HTTPS avec certificats SSL auto-signés. Virtual hosts configurés par domaine.
Base de données
PostgreSQL — Debian 12
Base de données principale utilisée par Odoo. Accès restreint au VLAN serveurs uniquement. Sauvegardes automatiques planifiées vers le NAS.
Stockage NAS
Synology DSM
Partages SMB/CIFS pour les dossiers utilisateurs, sauvegardes des VMs via Hyper Backup, synchronisation des profils AD. Quotas par utilisateur et par groupe.
ERP Métier
Odoo 17 Community
Gestion comptabilité, stocks, CRM et ressources humaines. Déployé sur VM Debian avec PostgreSQL dédié. Accessible via Nginx en HTTPS.

Étapes de déploiement

  • 01
    Conception du plan réseau — Définition des VLANs, plan d'adressage IP, schéma de routage et matrice de flux autorisés entre zones.
  • 02
    Déploiement pfSense — Installation et configuration du pare-feu : interfaces VLAN, règles de filtrage, NAT, DHCP par VLAN, DNS Resolver.
  • 03
    Active Directory & GPO — Promotion du contrôleur de domaine, création de l'arborescence OU, déploiement des GPO de sécurité (écran de veille, restrictions logicielles, mappage des lecteurs réseau).
  • 04
    NAS Synology — Configuration des partages SMB, intégration au domaine AD, activation des quotas, mise en place des tâches de sauvegarde automatiques.
  • 05
    Serveur web Nginx — Installation de Nginx sur Debian 12, configuration des virtual hosts, mise en place du reverse proxy vers Odoo et génération des certificats SSL.
  • 06
    PostgreSQL & Odoo — Déploiement de PostgreSQL, création de la base Odoo, installation et configuration d'Odoo 17 Community avec les modules comptabilité, stocks et CRM.
  • 07
    Tests et documentation — Vérification de la matrice de flux (ping, telnet), tests de connexion AD depuis les postes clients, accès Odoo via HTTPS, rédaction du dossier technique.

L'infrastructure couvre l'ensemble du périmètre SI d'une PME type. La segmentation VLAN assure l'isolation des flux sensibles, pfSense filtre tout le trafic inter-zones, et Odoo offre aux utilisateurs une solution ERP complète accessible de manière sécurisée via leur compte Active Directory.
